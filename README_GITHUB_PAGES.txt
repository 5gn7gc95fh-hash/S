CÓMO SUBIR A GITHUB PAGES:
1. Descarga el ZIP.
2. Descomprime.
3. Sube el archivo index.html a la raíz del repositorio.
4. Entra a Settings > Pages.
5. En Source elige Deploy from a branch.
6. En Branch elige main y /root.
7. Guarda y espera unos minutos.

NOTA:
Los datos se guardan en el navegador del dispositivo usando localStorage.
