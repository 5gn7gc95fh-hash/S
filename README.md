# PRECIOS SMARTPHONE

App web estática para GitHub Pages.

## Publicación rápida
1. Crear un repositorio en GitHub.
2. Subir `index.html` y `.nojekyll` a la raíz del repositorio.
3. Ir a Settings > Pages.
4. En Source elegir `Deploy from a branch`.
5. Branch: `main` / folder: `/root`.
6. Guardar y esperar unos minutos.

El archivo principal debe llamarse `index.html`.
